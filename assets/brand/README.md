# Handoff: شعار وهوية تطبيق «عوّاد»

## Overview
«عوّاد» تطبيق لبناء العادات. الهوية تقوم على فكرة واحدة: **الشدّة (ّ) فوق حرف الواو في الاسم تتحوّل إلى نبتة من ورقتين** — استعارة بصرية للنمو وتكرار العادة يوماً بعد يوم. هذه الحزمة تحوي الشعار الكامل (رمز + اسم عربي)، أيقونة التطبيق، وتطبيق الشعار على شاشتَي التطبيق (الافتتاح + الرئيسية).

## About the Design Files
الملفات في هذه الحزمة هي **مراجع تصميم مبنية بـ HTML** — نماذج تُظهر الشكل والسلوك المقصودَين، وليست كوداً إنتاجياً يُنسخ كما هو. المطلوب **إعادة إنشاء هذه التصاميم داخل بيئة كودبيس التطبيق** (React Native / Flutter / SwiftUI / أي إطار مستخدَم) باتباع أنماطه ومكتباته القائمة. إن لم توجد بيئة بعد، اختر الإطار الأنسب للمشروع ونفّذ التصاميم فيه.

الأصول المتجهة (SVG) في مجلد `assets/` **جاهزة للاستخدام المباشر** كما هي — انسخها إلى المشروع.

## Fidelity
**High-fidelity (hifi).** الألوان والخطوط والقياسات نهائية. أعد بناء الواجهة بدقة بكسلية باستخدام مكتبات الكودبيس، وأصول SVG المرفقة مباشرةً للشعار والأيقونة.

## Core Logo Construction (الأهم)

الشعار = كلمة **«عوّاد»** بخط عربي مدوّر، مع نبتة ورقتين مكان الشدّة فوق الواو.

**تقنية تثبيت النبتة فوق الواو (في نموذج الـ HTML):**
- الاسم يُقسّم إلى ثلاث قطع داخل عنصر `direction: rtl`: `عو` ثم النبتة (position:absolute) ثم `اد`.
- القطعة الحاوية للنبتة هي `عو` مع `position: relative`.
- النبتة `position: absolute; left: 15%; bottom: 100%; transform: translate(-50%, 58%)`.
- حجم النبتة نسبي للخط: `width: 0.5em; height: 0.52em` (لخط Baloo). القيم قد تحتاج معايرة بسيطة عند تغيير الخط.

> عند إعادة البناء في كود حقيقي، الأفضل تصدير الشعار الكامل (النص + النبتة) كـ **SVG وحيد ثابت** بدل إعادة بناء التموضع بالنص الحي — أدق وأثبت عبر المنصّات. أصول `assets/` تعطيك النبتة جاهزة؛ حوّل النص إلى منحنيات (outlines) في أداة فيكتور ثم ركّب النبتة فوق الواو مرة واحدة.

**النبتة (SVG):** viewBox `0 0 100 104`. ثلاثة أجزاء — ساق + ورقتان — مع عرقين رفيعين على الورقتين. الكود الكامل في `assets/awad-plant.svg` (ملوّنة) و`assets/awad-plant-mono.svg` (أحادية بـ currentColor).

## Screens / Views

### 1) شاشة الافتتاح (Splash)
- **Purpose:** أول ما يراه المستخدم عند فتح التطبيق.
- **Layout:** خلفية كحلية `#12161F` بملء الشاشة. المحتوى في المنتصف عمودياً وأفقياً، `flex-column`, `gap 26px`.
- **Components:**
  - أيقونة النبتة داخل مربّع كحلي مرفوع `#1B2130` (104×104، radius 26، حدّ `rgba(160,205,95,.16)`)، النبتة بداخله 62×64.
  - الاسم «عوّاد» بخط Baloo 600، حجم ~64px، لون كريمي `#F4F2EC`، والنبتة الملوّنة فوق الواو.
  - سطر شعاري: «ازرع عادةً جديدة كل يوم» — Tajawal 500، 15px، لون `#8A93A0`.
  - عناصر النظام: Dynamic Island، شريط حالة كريمي، مؤشّر Home سفلي.

### 2) الشاشة الرئيسية — «اليوم» (Home / Today)
- **Purpose:** عرض عادات اليوم وتقدّمها وتسجيل الإنجاز.
- **Layout:** خلفية فاتحة `#F6F4EE`. من الأعلى: ترويسة → تحية → بطاقة تقدّم → قائمة عادات → شريط تنقّل سفلي ثابت.
- **Components:**
  - **الترويسة:** يمين — أيقونة صغيرة (مربّع كحلي 36px radius 11 + نبتة) واسم «عوّاد» (Baloo 600، 21px، كحلي). يسار — شارة تتابع pill خضراء فاتحة `#EEF4E1` فيها نبتة صغيرة + «١٢ يوم» (Cairo 700، 13px، `#4E7A2A`).
  - **التحية:** «مساء الخير، محمد» (Tajawal 800، 23px) + التاريخ «الخميس، ٥ يوليو» (Tajawal 500، 13.5px، `#7A8290`).
  - **بطاقة التقدّم:** خلفية كحلية `#12161F`، radius 20. حلقة تقدّم دائرية (SVG، قطر 60، مسار `rgba(255,255,255,.14)`، تقدّم `#9FCE57`، سماكة 3.4) بنسبة ٥٠٪ في المنتصف. نص: «عادات اليوم» (Tajawal 700، 16px، كريمي) + «أكملت ٢ من ٤ — استمر في النمو» (Tajawal 500، 12.5px، `#9BA3B0`).
  - **قائمة العادات (٤ عناصر):** بطاقات بيضاء radius 16، حدّ `#ECEAE0`، padding 13/14.
    - «قراءة ١٠ صفحات» — منجَزة: نبتة صغيرة + «٧ أيام متتالية»، وزر دائري أخضر `#6FA83C` بعلامة ✓ بيضاء.
    - «شرب ٨ أكواب ماء» — قيد التقدّم: شريط تقدّم (خلفية `#EDEBE1`، تعبئة `#8FBF44` بنسبة 62%) + «٥/٨» (Cairo 700).
    - «تمرين الصباح» — غير منجَزة: «١٢ يوماً متتالياً» + دائرة فارغة بحدّ `#D6D3C6`.
    - «تأمّل ٥ دقائق» — غير منجَزة: «٤ أيام متتالية» + دائرة فارغة.
  - **شريط التنقّل السفلي:** أربع تبويبات (اليوم/العادات/التقدّم/حسابي) بأيقونات خطّية. النشط «اليوم» بلون `#5E8A31`، البقية `#A6ABB4`. خلفية بيضاء بحدّ علوي `#ECEAE0`.

## Interactions & Behavior
هذه نماذج ثابتة (static)؛ السلوك المقصود عند التنفيذ:
- **الافتتاح:** يظهر ~1.5–2 ثانية ثم ينتقل للرئيسية (أو شاشة الدخول). مناسب إضافة ظهور تدريجي/نمو خفيف للنبتة.
- **تسجيل عادة:** الضغط على الدائرة يبدّلها بين منجَزة/غير منجَزة، مع تحديث حلقة التقدّم ونسبتها فورياً.
- **شارة التتابع (streak):** تزيد يوماً عند إكمال كل عادات اليوم.
- **التنقّل السفلي:** يبدّل بين الأقسام؛ التبويب النشط يأخذ اللون الأخضر.
- **الأرقام:** كلها بالأرقام العربية الهندية (٠١٢٣…). احرص على تنسيق الأرقام حسب اللغة.
- **الاتجاه:** كل الواجهات RTL.

## State Management (مقترح)
- `habits[]`: لكل عادة `{ id, title, type: 'check'|'progress', current, target, streak, done }`.
- `todayProgress`: محسوبة من نسبة العادات المنجَزة (تغذّي الحلقة والنسبة).
- `streak`: عدّاد التتابع العام.
- `activeTab`: التبويب الحالي في الشريط السفلي.
